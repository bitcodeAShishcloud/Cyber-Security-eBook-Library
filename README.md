# Cyber Security eBook Library

A comprehensive, open-source collection of the best books, guides, and standards across all major domains of cybersecurity. This project is designed to help students, professionals, and organizations quickly find authoritative resources for learning, reference, and implementation.

## Features
- Modern, responsive web UI for browsing and searching books by domain
- Direct download links to official and open-source resources
- Curated, high-quality content for every major cybersecurity discipline
- Easy to extend: just update `book.json` to add new books or domains

## Usage
1. Open `Cyber Security eBook Library.html` in your browser (use a local web server for best results).
2. Browse or search for books by domain or title.
3. Click the download button to access the official resource.

## How to Add or Update Books
- Edit `book.json` and add new domains or books as needed.
- The UI will automatically reflect your changes.
